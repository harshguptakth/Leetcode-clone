# 🚀 LeetCode Clone – Full Stack Coding Platform

A full-stack LeetCode-style coding platform built using modern web technologies.  
This project allows users to practice coding problems, submit solutions, and interact with AI-based doubt solving.

---

## 🛠 Tech Stack

### 🎨 Frontend
- React.js (Vite)
- Redux Toolkit
- Axios
- Tailwind CSS / DaisyUI

### ⚙️ Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- Redis

---

## ✨ Features

- 🔐 User Authentication (Login / Signup)
- 🧠 Solve Coding Problems
- 📜 Submission History Tracking
- 🎥 Video Solution Section
- 🤖 AI Doubt Solver Chat
- 🛠 Admin Panel for Problem Management
- ⚡ Fast API with Redis Caching

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/harshguptakth/Leetcode-clone.git
cd Leetcode-clone
2️⃣ Install Backend Dependencies
cd backend
npm install
3️⃣ Install Frontend Dependencies
cd ../frontend
npm install
4️⃣ Run Backend Server
cd ../backend
npm run dev

Backend runs on:

http://localhost:3000
5️⃣ Run Frontend
cd ../frontend
npm run dev

Frontend runs on:

http://localhost:5173
